# Congress Trading Copilot

Personal AI-powered congressional trade monitoring and execution system.

## Goals

- Monitor congressional disclosures
- Apply delay filter
- Generate Congress Score
- Generate Consensus Score
- Generate Trade Quality Score
- Send Telegram alerts
- Execute trades via Alpaca
- Provide ChatGPT access through MCP
